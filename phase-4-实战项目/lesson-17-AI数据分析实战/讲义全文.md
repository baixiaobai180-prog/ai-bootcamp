# 第17课 · AI数据分析实战

![配图](assets/images/lesson-17-data-analysis.jpg)


## 一、AI能做哪些数据分析

现在的AI具备很强的数据分析能力，你不需要会写SQL或Python，也不需要会用Excel的高级功能。

**AI能做的事情：**
1. **理解数据** — 上传Excel/CSV，AI自动分析数据概要
2. **回答问题** — 用自然语言提问，AI给出数据答案
3. **生成图表** — AI写代码生成可视化图表
4. **撰写报告** — AI分析数据并写出分析报告

## 二、让AI分析Excel/CVS数据

**方法1：上传文件给ChatGPT/Claude**
- 直接把Excel文件拖进对话框
- AI会读取并分析
- 你可以问各种问题

**方法2：复制数据直接提问**
- 把数据复制粘贴
- 问任何分析相关的问题

## 三、提问示例

**好的数据提问：**
- "这个数据集中，哪个产品类别的销售额最高？请用柱状图展示"
- "分析最近3个月的销售趋势，是否有季节性规律？"
- "把所有用户按消费金额分成高/中/低三档，描述各档的特点"
- "在这个数据中，有没有发现异常值？可能是数据录入错误"

## 四、推荐工具

- **ChatGPT + Code Interpreter** — 直接上传数据分析
- **Claude + Artifacts** — 上传数据+生成图表
- **Julius AI** — 专门的数据分析AI
- **通义千问** — 国产，支持上传Excel
