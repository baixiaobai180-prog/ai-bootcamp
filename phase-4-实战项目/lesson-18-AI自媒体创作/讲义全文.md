# 第18课 · AI+自媒体创作全流程

![配图](assets/images/lesson-18-content-creation.jpg)


## 一、AI辅助选题

**用AI发现好选题：**
```
Prompt：我是一个科技类自媒体博主，最近AI行业有这些热点[列出3-5个]。
请帮我分析哪些选题最容易出爆款，给出5个选题建议并说明理由。
```

**用AI做竞品分析：**
```
Prompt：请分析[某账号]最近10篇爆款内容，总结其选题规律、
标题特点、内容结构和互动技巧。
```

## 二、AI写爆款文案

**标题生成：**
```
请为以下文章生成10个爆款标题，要求包含数字、对比、悬念元素。
文章主题：[你的主题]
```

**正文结构模板：**
- **SCQA模型**：情境→冲突→问题→答案
- **AIDA模型**：注意→兴趣→欲望→行动
- **黄金圈**：为什么→怎么做→做什么

## 三、全流程实战

以"AI学习班第XX课"为例：

1. **选题**：让AI分析近期AI热门话题
2. **标题**：AI生成10个标题，选最好的
3. **大纲**：AI生成内容结构
4. **正文**：AI分部分写作，人工调整
5. **封面**：AI绘画生成配图
6. **摘要**：AI提取金句和摘要
7. **多平台**：AI适配不同平台格式
