# 第16课 · AI自动化工作流设计

## 一、什么是自动化工作流

想象你有这样一个需求：
> 每天早上9点，自动搜集行业新闻，让AI分析重点，然后推送到你的微信群里。

这就是一个典型的自动化工作流。

**工作流的三个要素：**
1. **触发器** — 什么时候开始（定时/收到邮件/文件更新）
2. **处理节点** — AI分析、数据处理、判断决策
3. **输出节点** — 发送通知、保存文件、更新数据库

## 三、实战：搭建"每日AI行业简报"

工作流设计：
```
1. 触发器：每天早上8:00
2. 获取数据：爬取指定网站/公众号（或RSS）
3. AI处理：提取关键信息、写摘要
4. AI整理：按重要度排序、分类
5. 输出：推送到飞书/微信/邮箱
```

## 四、更多自动化场景

- **日报自动生成**：收集工作记录→AI写日报→发邮件
- **简历筛选**：收到简历→AI打分→分类归档
- **竞品监控**：定时检查竞品动态→AI差异分析→生成报告
- **客服分流**：用户咨询→AI分类→分配对应客服
