# 动手练习 · 第14课

## 练习一：判断要不要微调（难度：⭐⭐）

**任务：** 判断以下场景是否适合微调：

1. 我想让AI回答关于我家宠物的问题
2. 我想让AI用我公司的产品手册来回答客户问题
3. 我想让AI永远用"网状结构"来写文章（第一段是结论，后面展开）
4. 我想让AI翻译成文言文
5. 我想让AI分析我的excel数据

**参考答案：** 1→RAG, 2→RAG, 3→微调, 4→微调, 5→Prompt

## 练习二：数据集设计（难度：⭐⭐⭐）

**任务：** 假设你要微调一个AI，让它"用猫的语气说话"。请设计5条训练数据。

**格式：**
```
{
  "input": "你好",
  "output": "喵~今天有鱼吃吗？"
}
```
