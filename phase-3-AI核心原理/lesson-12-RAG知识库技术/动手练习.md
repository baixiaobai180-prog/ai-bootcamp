# 动手练习 · 第12课

## 练习一：体验RAG（难度：⭐⭐）

**任务：** 用AnythingLLM或Dify搭建一个简单的知识库。

**步骤：**
1. 下载安装AnythingLLM
2. 选择AI模型（可以选免费的）
3. 导入3-5个你自己的文档
4. 问几个和文档相关的问题

## 练习二：对比实验（难度：⭐⭐⭐）

**任务：** 做一个"有RAG"vs"无RAG"的对比。

**步骤：**
1. 准备一份关于某个小众话题的文档
2. 不用RAG，直接问ChatGPT/Claude同一个问题
3. 用RAG（AnythingLLM）问同一个问题
4. 对比回答质量和准确度

## 练习三：场景想象（难度：⭐）

**任务：** 列出3个你生活中可以用RAG的场景。

示例：
"我可以把买房合同相关的问题做成一个知识库，让AI帮我解读合同条款。"
