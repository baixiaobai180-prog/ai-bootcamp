# 第12课 · RAG技术：让AI拥有你的专属知识库

## 课件大纲

### 一、纯LLM的局限（10分钟）
1. 知识截止于训练日期（如GPT-4知识截止于2024年）
2. 幻觉（Hallucination）：胡说八道
3. 不知道你的私人数据
4. 数据安全合规问题

### 二、RAG是什么？（15分钟）
- RAG = Retrieval-Augmented Generation（检索增强生成）
- 工作流程：用户问题 → 检索相关文档 → 把文档拼进Prompt → AI回答

### 三、向量嵌入（15分钟）
- 嵌入（Embedding）：把文本变成向量
- 语义搜索：找"意思相近"而不是"关键词相同"
- 余弦相似度：衡量两个向量多接近

### 四、向量数据库（10分钟）
- 存向量的专门数据库
- 常用：ChromaDB、Pinecone、Milvus

### 五、完整RAG流程（20分钟）
```
文档 → 分割(Chunking) → 向量化(Embedding) → 存向量数据库
用户提问 → 向量化 → 向量数据库检索相似内容 → 
把检索到的内容+用户问题拼成Prompt → LLM回答
```

### 六、实战工具（20分钟）
- Dify：开源RAG平台
- AnythingLLM：极简桌面应用
- 飞书知识库

### 七、优化技巧（10分钟）
- Chunk大小选择
- 检索策略（Top-K）
- 重排序（Re-ranking）
