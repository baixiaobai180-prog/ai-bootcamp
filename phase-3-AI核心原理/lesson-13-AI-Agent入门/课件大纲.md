# 第13课 · AI Agent入门：让AI自主完成任务

![配图](assets/images/lesson-13-ai-agent.jpg)


## 课件大纲

### 一、什么是AI Agent（15分钟）
- Agent = AI + 工具 + 自主决策
- 感知 → 思考 → 行动 → 观察 → 再思考……
- LLM是"大脑"，工具是"手和脚"

### 二、Agent三大组件（20分钟）
1. **LLM大脑** — 规划、决策、推理
2. **工具** — 搜索、计算、文件、API
3. **记忆** — 短期记忆（上下文）+ 长期记忆（存外部）

### 三、Function Calling（15分钟）
- LLM输出"调哪个函数"的请求
- 系统执行函数，返回结果给LLM
- LLM基于结果继续处理

### 四、Agent框架概览（15分钟）
- LangChain：最流行的框架
- AutoGen：微软出品，多Agent协作
- CrewAI：角色扮演式Agent团队

### 五、常见Agent模式（15分钟）
- ReAct：推理+行动循环
- Plan-and-Execute：先规划再执行
- Multi-Agent：多个Agent合作

### 六、实战思路（10分钟）
- 简单Agent：自动搜索资料
- 中等Agent：自动写报告
- 复杂Agent：客服系统
