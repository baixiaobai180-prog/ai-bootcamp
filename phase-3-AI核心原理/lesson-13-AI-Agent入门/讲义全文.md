## 一、什么是AI Agent

AI Agent（智能体）是当前AI领域最热的方向之一。

**它的核心区别在于：** 普通AI是"你问它答"，而Agent是"给它一个目标，它自己完成任务"。

**Agent的工作循环：**
```
用户给目标 → AI思考该做什么 → AI调用工具 → 
AI观察结果 → AI调整策略 → 继续行动 → 直到完成
```

**一个形象的比喻：**
> 普通AI就像你的助理，你告诉它"帮我查一下北京明天的天气"，它就照做。
> Agent就像项目经理，你说"帮我安排一次北京出差"，它会自己订机票、订酒店、查天气、排日程，遇到问题自己想办法。

## 二、Agent三大组件

### 1. LLM大脑
Agent的"思考器官"。负责理解目标、制定计划、分析结果。

### 2. 工具
Agent的"行动器官"。包括：
- 网页搜索
- 计算器
- 文件读写
- API调用
- 数据库查询
- 代码执行

### 3. 记忆
- **短期记忆**：当前任务的上下文（类似LLM的上下文窗口）
- **长期记忆**：跨任务的持久化存储

## 三、ReAct模式

ReAct = Reasoning + Acting（推理+行动）

**工作流程：**
1. Thought（思考）：我该做什么？
2. Action（行动）：调用某个工具
3. Observation（观察）：工具返回了什么结果
4. 回到第1步，直到问题解决

## 四、Agent能做哪些事

- **研究助理**：自动搜索资料、整理、写报告
- **客服机器人**：查订单、退换货、回答FAQ
- **数据分析**：查询数据库、做图表、分析趋势
- **自动化运营**：定时发文章、监控数据、发送报告

## 五、用Agent的方式思考

学完了这一课，你可以开始用"Agent思维"思考问题：
> "这个问题能不能交给AI Agent自动完成？"

把日常重复性的工作记录下来，那就是你第一个Agent的潜在任务。
