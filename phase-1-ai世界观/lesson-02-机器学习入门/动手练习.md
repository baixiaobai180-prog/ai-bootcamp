# 动手练习 · 第02课

## 练习一：给下面任务分类（难度：⭐）

判断每个任务适合哪种机器学习方式（监督学习/无监督学习/强化学习）：

1. 判断信用卡交易是否欺诈
2. 根据用户历史浏览记录推荐商品
3. AI学会玩《超级马里奥》
4. 将新闻文章自动分成体育、娱乐、科技等类别
5. 预测明天股票涨跌
6. 自动识别照片中的人脸是谁
7. AI在围棋比赛中击败人类选手
8. 电商把所有用户分成"高消费/中等/低消费"三组

**参考答案：** 1监督 2监督 3强化 4无监督 5监督 6监督 7强化 8无监督

## 练习二：说出你的数据（难度：⭐⭐）

**任务：** 想一个你想让AI帮你解决的问题，然后回答：

1. 这个问题需要什么数据？
2. 数据从哪里来？
3. 用哪种学习方式？
4. 怎么知道AI学好了？

**示例：**
```
问题：自动判断一道菜好不好吃
数据：一万张菜品的图片 + 每张图上标了好吃/不好吃
方式：监督学习（分类）
判断标准：准确率达到90%以上
```

## 练习三：找找你身边的机器学习（难度：⭐）

**任务：** 今天找3个"背后用了机器学习"的产品/服务，记录：
- 产品叫什么
- 它做了什么
- 你觉得它用的哪种学习方式

## 练习四：用ChatGPT学机器学习（难度：⭐⭐）

**任务：** 打开ChatGPT或Claude，问它下面这个问题，看看它怎么回答：

> "用最简单的话解释机器学习是什么，举一个生活例子，像在跟一个10岁小孩解释。"

然后把你得到的回答记录下来，和讲义里的内容对比一下。
