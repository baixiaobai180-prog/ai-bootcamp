![机器学习概念图](assets/images/lesson-02-machine-learning.jpg)


## 一、传统编程 vs 机器学习

在讲机器学习之前，我们先想想传统编程是怎么工作的。

**传统编程的工作方式：**
```
人写规则 → 交给计算机 → 计算机按规则执行
```

比如我们想写一个判断水果的程序：
```
规则1：如果是圆的、红色的 → 苹果
规则2：如果是弯的、黄色的 → 香蕉
规则3：如果是圆的、橙色的 → 橙子
......
```
你要把所有规则一条条写清楚，计算机才能工作。如果遇到没见过的水果，程序就傻了。

**机器学习的工作方式：**
```
给大量数据 → 机器自己总结规律 → 得到一个模型
```

还是判断水果，我们不给规则，而是给一万张标好了名字的水果照片。机器自己"看"这些照片，自己总结出"红色圆形的是苹果"这个规律。

这就像教小孩认动物——你不用跟他讲"哺乳动物纲食肉目犬科"这些生物学分类，你只要多指几次"这是狗、这是猫"，他自然就学会了。

**一句话：机器学习就是让计算机从数据中自己学习规律，而不是靠人一条条写规则。**

## 二、三种主流学习方式

### 1. 监督学习（Supervised Learning）

**最常用的方式。** 就像老师给学生布置有标准答案的作业。

你给AI一堆"问题+答案"的组合，让它学会看到问题就能答出答案。

**分两类：**

**① 分类（Classification）— 判断类别**
- 例子：这张图片是猫还是狗？
  输入：图片 → 输出：猫/狗
- 例子：这封邮件是垃圾邮件吗？
  输入：邮件内容 → 输出：是/否

**② 回归（Regression）— 预测数值**
- 例子：明天温度多少度？
  输入：历史天气数据 → 输出：26.5°C
- 例子：这套房子值多少钱？
  输入：面积、位置、楼层 → 输出：350万

### 2. 无监督学习（Unsupervised Learning）

**不给答案，让机器自己找规律。** 就像让一群小孩自己组队玩，谁和谁玩得来是他们的自由组合。

**典型应用：聚类（Clustering）**
- 电商把用户分群：喜欢买运动鞋的、喜欢买化妆品的、喜欢买电子产品的……
- 新闻自动分类：体育新闻、娱乐新闻、科技新闻……
- 你不用告诉机器"体育新闻的特征是什么"，它自己会从文章相似度中分组

### 3. 强化学习（Reinforcement Learning）

**通过试错和奖励来学习。** 就像训练小狗——做对了给零食，做错了不理它。

**特色：**
- 没有"标准答案"，只有"行动"和"反馈"
- 在不断的尝试中优化策略
- 适合做决策、玩游戏、控制机器人

**经典例子：**
- AlphaGo下围棋：赢了是奖励，输了是惩罚
- 自动驾驶：安全驾驶得奖励，撞车得惩罚
- 游戏AI打《王者荣耀》：推塔得分是奖励

### 三种方式对比

| 方式 | 需要标签？ | 原理 | 典型应用 |
|------|-----------|------|---------|
| 监督学习 | ✅ 需要 | 给标准答案学 | 识别猫狗、垃圾邮件过滤 |
| 无监督学习 | ❌ 不需要 | 自己找规律 | 用户分群、新闻分类 |
| 强化学习 | ❌ 不需要 | 试错得奖励 | 下棋、自动驾驶、游戏 |

## 三、机器学习的三大要素

### 1. 数据（Data）— 燃料
- 没有数据，一切AI都是空谈
- 数据越多、质量越高，模型效果越好
- "垃圾进，垃圾出"（Garbage In, Garbage Out）
- 数据清洗和标注是非常重要的工作

### 2. 模型（Model）— 引擎
- 算法的架构设计
- 不同的任务用不同的模型
- 好的模型设计能让学习效率更高

### 3. 算力（Computing Power）— 动力
- 训练大模型需要大量计算
- GPU（显卡）是主力，比CPU快几十倍
- 现在训练一个GPT-4级别的模型，成本在数亿美元级别

## 四、生活中的机器学习

| 场景 | 用的什么学习方式？ |
|------|------------------|
| 抖音推荐"你可能喜欢" | 监督学习（预测你的喜好） + 无监督学习（用户分群） |
| 淘宝"猜你喜欢" | 同上 |
| 地图导航避开拥堵 | 监督学习（预测路况） |
| 人脸解锁 | 监督学习（分类：是你/不是你） |
| 垃圾短信拦截 | 监督学习（分类：垃圾/正常） |
| 天气预报 | 监督学习（回归：预测温度/降雨概率） |

## 五、本课小结

- 机器学习让机器从数据中自己学规律，而不是靠人写规则
- 三种学习方式：监督学习（有答案）、无监督学习（无答案）、强化学习（试错）
- 三大要素：数据、模型、算力
- 生活中到处是机器学习，只是你不知道

> 💡 **金句**：机器学习不是魔法，是用数学从数据中找规律。
