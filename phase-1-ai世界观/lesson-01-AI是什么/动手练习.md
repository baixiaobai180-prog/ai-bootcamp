# 动手练习 · 第01课

## 练习一：找AI（难度：⭐）

**任务：** 记录你一天中接触到AI的场景。

**要求：**
1. 从起床到睡觉，列出至少10个用到了AI的场景
2. 判断每个场景属于哪种AI能力（感知/思考/行动）
3. 标注哪些是强AI、哪些是弱AI

**示例：**
```
时间: 08:00
场景: 用Face ID解锁手机
AI能力: 感知（人脸识别）
类型: 弱AI
```

## 练习二：图灵测试小实验（难度：⭐⭐）

**任务：** 和ChatGPT/Claude对话5分钟，看看你能不能分辨出它是AI。

**步骤：**
1. 打开ChatGPT或Claude
2. 和它聊任何一个话题（比如"推荐一部好看的电影"）
3. 观察它的回答：
   - 它听起来像人吗？
   - 有没有那种"AI感"？
   - 你觉得它通过图灵测试了吗？

**思考：** 如果对方也是真人，你能分辨吗？

## 练习三：AI vs 人类（难度：⭐⭐）

**任务：** 下面这些事，你觉得AI做得好还是人类做得好？在每项后面打✓（AI胜）或✗（人类胜）或⚖（打平）

| 任务 | AI胜？ | 人类胜？ | 打平？ |
|------|--------|---------|-------|
| 计算圆周率第1000位 | | | |
| 创作一首感人的诗 | | | |
| 认出照片里的朋友 | | | |
| 下围棋 | | | |
| 安慰一个伤心的人 | | | |
| 翻译外语 | | | |
| 开车 | | | |

**参考答案：** 计算→AI胜；诗→人类胜；认人→人类胜；围棋→AI胜；安慰→人类胜；翻译→⚖打平；开车→⚖打平（还在发展中）

## 练习四：写一段话给10年后的AI（难度：⭐）

**任务：** 写一段50-100字的话，写给你认为10年后（2036年）的AI。

**提示：** 你想对那时候的AI说什么？有什么期待？有什么担心？

## 练习五：AI新闻追踪（难度：⭐⭐）

**任务：** 找一篇关于AI的新闻（可以用搜索引擎搜"AI 2025"或"人工智能 最新"），然后回答：
1. 这篇新闻在说什么？
2. 它说的是强AI还是弱AI？
3. 你觉得这个新闻可信吗？为什么？
