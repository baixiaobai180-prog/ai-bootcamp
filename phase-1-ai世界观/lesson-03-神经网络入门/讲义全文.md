# 第03课 · 神经网络入门：AI的大脑长什么样

![神经网络结构图](assets/images/lesson-03-neural-network.jpg)


## 一、从人类大脑说起

上节课我们讲了机器学习，但有个问题没回答：**机器学习里的"模型"到底是什么？**

当今AI世界，最主流的答案就是**神经网络**（Neural Network）。

这个名字不是白起的——它确实受到了人类大脑的启发。

**人脑大概长这样：**
- 有大约860亿个神经元（神经细胞）
- 每个神经元通过"突触"和上千个其他神经元相连
- 当神经元收到足够强的信号时，它会"激活"，把信号传给下一个神经元

**人工神经元就是模仿这个机制：**
- 接收多个输入信号
- 每个信号有不同的"重要性"（权重）
- 综合判断是否"激活"
- 如果激活了，就把信号传给下一层

**打个比方：**
> 神经元就像公司的员工。每个员工收到上级的指示（输入），结合自己的判断（权重），决定要不要继续往下传。一层一层的员工合作，最终做出一个决策（输出）。

## 二、感知机：最基本的神经网络

**感知机（Perceptron）** 是1957年提出的最早的神经网络——它只有一个神经元。

它的工作流程：
1. 接收多个输入（比如 x₁=0.8, x₂=0.3）
2. 每个输入乘以对应的权重（比如 w₁=0.5, w₂=-0.2）
3. 把所有乘积加起来 + 偏置（bias）
4. 如果总和大于某个阈值 → 输出1（激活）
5. 如果小于阈值 → 输出0（不激活）

**一个很形象的比喻：**
> 你要决定今天去不去海边。输入因素：天气好不好（x₁，0~1）、有没有空（x₂，0~1）、交不交通告（x₃，0~1）。权重：天气占80%（w₁=0.8），有空占50%（w₂=0.5），交通告占-60%（w₃=-0.6）。综合得分超过0.5就去，否则不去。

感知机虽然简单，但它有个致命的局限：**只能解决线性可分的问题**（比如AND、OR），解决不了XOR（异或问题）。这就催生了多层神经网络。

## 三、多层神经网络：让网络更深

把多个感知机叠在一起，就形成了**多层神经网络**：

```
输入层 → 隐藏层① → 隐藏层② → ... → 输出层
```

**各层的作用：**
- **输入层**：接收原始数据（图片的像素、文字的Token、音频的波形）
- **隐藏层**：层层提取特征（从简单到复杂）
- **输出层**：产出最终结果（类别、数值、文本）

**"深度"学习**的"深度"就是指隐藏层的数量。层数越多，模型越大，能学到的模式越复杂。

早期的神经网络只有1-2个隐藏层，现在的GPT有几百层、几千亿个参数。

## 四、激活函数：给神经网络注入灵魂

如果没有激活函数，会发生什么？

**答案是：无论叠多少层，都等价于一层。**

因为线性变换的叠加还是线性变换。就像无论你把多少把尺子接在一起，它还是只能画直线。

**激活函数的作用就是引入非线性**，让神经网络能模拟复杂的关系。

**常见的激活函数：**

| 函数 | 长什么样 | 特点 |
|------|---------|------|
| **ReLU** | 负数→0，正数→本身 | 简单好用，最常用 |
| **Sigmoid** | 挤压到0~1之间 | 适合做概率输出 |
| **Tanh** | 挤压到-1~1之间 | 输出有正有负 |

现在绝大多数神经网络都用ReLU（修正线性单元），因为它计算简单、效果好。

## 五、神经网络如何学习

这是最关键的问题。一个神经网络如何从"瞎猜"变成"很准"？

### 步骤1：前向传播（Forward Propagation）
把输入数据传进网络，逐层计算，最终得到一个输出。

一开始，因为权重是随机初始化的，输出肯定一塌糊涂。

### 步骤2：计算损失（Loss Function）
比较输出和正确答案差多少。差得越多，"损失值"越大。

比如识别手写数字"7"，模型输出了"有20%可能是7，60%可能是1，10%可能是4..."，而正确答案是"100%是7"——这时损失值就很大。

### 步骤3：反向传播（Backpropagation）
从输出层开始，反向计算每个权重对损失值的"责任"有多大。

**简单理解：** 谁的权重导致了错误的判断，谁就要被调整。

### 步骤4：梯度下降（Gradient Descent）
根据反向传播的结果，逐层调整每个权重，让损失值变小一点。

**登山类比：** 你站在山上想下山（想减小损失值），你会朝哪个方向走？当然是朝着最陡的下降方向走。每一步都找最陡的方向，最终就能走到山谷。

你重复步骤1-4，一遍又一遍。每一次迭代，模型都会变好一点点。

**训练一个模型就像训练一个小狗：**
1. 它第一次肯定做错（前向传播）
2. 你知道它做错了（计算损失）
3. 你告诉它哪里错了（反向传播）
4. 它调整自己的行为（梯度下降）
5. 重复直到它做对

## 六、直观理解：识别手写数字的过程

假设我们要识别手写数字"7"：

**第一层神经元**：看单个像素。有的神经元发现"这里有个黑点"，有的发现"这里有个斜线"。

**中间层**：组合上一层的发现。有些神经元认出"有一条横线在上方"，有些认出"有一条斜线从左到右"。

**最后一层**：综合所有特征。"上方有横线 + 有斜线 → 大概率是7"。

这就是**层次化特征提取**——从简单到复杂、从具体到抽象。

## 七、本课小结

- 神经网络受人类大脑启发，由大量"神经元"组成
- 单个神经元 = 输入×权重 + 偏置 → 激活 → 输出
- 多层网络通过"深度"提取层次化特征
- 激活函数引入非线性，是网络能学习复杂模式的关键
- 训练过程：前向传播 → 算损失 → 反向传播 → 梯度下降 → 循环

> 💡 **一句话总结：神经网络就是一个超级大的数学函数，通过反复调参数来拟合数据中的模式。**
