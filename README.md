# 🎓 零基础AI学习班

> 从完全小白到能独立使用AI解决实际问题的完整课程体系

## 📌 课程定位

**面向人群：** 零编程基础、零AI知识的普通用户
**学习目标：** 理解AI原理 → 熟练使用AI工具 → 能搭建个人AI应用
**课时总数：** 20课时（每课时约1-2小时）
**学习周期：** 推荐8-10周，每周2-3课时
**前置要求：** 能正常上网、有基本的电脑操作能力

## 🗺️ 课程体系

```
第一阶段：AI世界观（第1-4课） → 建立AI认知框架
         ↓
第二阶段：动手玩AI（第5-10课） → 熟练使用主流AI工具
         ↓
第三阶段：AI核心原理（第11-14课） → 理解背后技术原理
         ↓
第四阶段：实战项目（第15-18课） → 独立完成AI项目
         ↓
第五阶段：AI素养与未来（第19-20课） → 建立AI价值观与职业规划
```
