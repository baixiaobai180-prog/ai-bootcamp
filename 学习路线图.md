# 🗺️ 学习路线图

根据你的情况选择不同的学习路径：

## 路线一：快速上手（2周）
适合：想先用起来的职场人士

```
第05课 → 第06课 → 第10课 → 第09课
```

只学使用层面，先会用再理解原理。两周内就能用AI提效。

## 路线二：系统学习（8周）
适合：想系统掌握的小白（推荐）

```
第01课 → 第02课 → 第03课 → 第04课 → 
第05课 → 第06课 → 第07课 → 第08课 →
第11课 → 第12课 → 第15课 →
第19课 → 第20课
```

每周2课，8周完成。兼顾理解和使用。

## 路线三：全栈掌握（10周）
适合：想把AI变成一项技能

全部20课，每周2课。
从原理到实践到项目，一条龙学完。

## 路线四：自学达人版
适合：有特定需求的人

- 我是创作者：第07课 → 第08课 → 第18课
- 我是上班族：第05课 → 第06课 → 第10课 → 第16课
- 我想转行AI：第01课 → 第11课 → 第12课 → 第13课 → 第14课 → 第20课
- 我是学生：全部按顺序学

## 学习建议

1. **动手是第一原则** — 每课的动手练习必须做
2. **不要跳阶段** — 第一阶段打好基础很重要
3. **多提问** — 用AI学AI，遇到不懂的直接问ChatGPT/Claude
4. **记笔记** — 整理自己的AI学习笔记
5. **找伙伴** — 组队学习效果更好
6. **复利效应** — 每课都产出一个小作品
