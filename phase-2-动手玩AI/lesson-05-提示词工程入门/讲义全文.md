# 第05课 · 提示词工程入门：和AI对话的正确姿势

## 一、什么是Prompt？

**Prompt（提示词）** 就是你输入给AI的指令或问题。

在过去的计算机时代，你要和电脑沟通，得学编程语言——Java、Python、SQL……但现在，你只需要**说人话**。

但这并不意味着你随便说一句，AI就能完美理解你。

> 和AI对话就像和一个非常聪明、但从不主动问你在想什么的外国助手打交道。你需要把话说清楚。

## 二、好Prompt vs 差Prompt

直接看对比：

**差Prompt：**
```
用户：帮我写篇文章
AI：好的，请问您想要什么主题的文章？
```

**好Prompt：**
```
你是资深科技记者。
请写一篇800字的科普文章，主题是"神经网络如何工作"。
目标读者：完全不懂技术的普通人。
要求：用比喻和例子，避免专业术语。
结构：开头吸引人 → 通俗解释 → 生活类比 → 总结
```
AI会直接给出一篇高质量的文章。

**差Prompt的问题在于：**
- 太宽泛，AI不知道你的需求
- AI不得不"猜"，猜错了你就觉得AI不行
- 浪费了多次对话来"纠正"AI

## 三、Prompt四大要素

### 要素1：角色（Role）
告诉AI它应该扮演什么角色。

**示例：**
```
❌ "解释量子力学"
✅ "你是一位物理学教授，擅长用生活例子解释复杂的物理概念。请用10岁小孩能听懂的语言解释量子力学。"
```

**常见角色：**
- 你是一位资深程序员
- 你是一名英语教师
- 你是一位营销专家
- 你是一名心理咨询师
- 你是一位历史学家

### 要素2：任务（Task）
告诉AI具体要做什么，越具体越好。

**示例：**
```
❌ "优化这段话"
✅ "把下面这段产品介绍改写成小红书种草风格，突出产品性价比，增加emoji，控制在200字以内：……"
```

### 要素3：格式（Format）
告诉AI你想要的输出格式。

**示例：**
```
❌ "给我几个新年计划建议"
✅ "给我5个2026年新年计划建议，以表格形式呈现：计划名称 | 具体做法 | 预期效果"
```

**常见格式：**
- 表格
- 列表（序号/项目符号）
- Markdown
- 代码块
- 邮件正文
- 对话脚本

### 要素4：约束（Constraint）
告诉AI什么不能做、注意什么。

**示例：**
```
❌ "写封求职信"
✅ "写一封求职信，要求：不超过300字，语气自信但不傲慢，突出我的数据分析经验，不提我的英语水平"
```

## 四、进阶Prompt技巧

### 技巧1：思维链（CoT）——让AI一步步想

当问题比较复杂时，让AI逐步推理：

```
❌ "小明买了3个苹果，每个2元，又买了2个橙子，每个3元。他一共花了多少钱？"
✅ "小明买东西，让我们逐步计算：
    1. 苹果3个，每个2元 → 3×2=6元
    2. 橙子2个，每个3元 → 2×3=6元
    3. 总共：6+6=12元
所以小明明一共花了12元。"
```

加了"让我们逐步计算"后，AI的准确率会大幅提升。

### 技巧2：Few-shot（少样本示例）

在提问前先给1-3个例子，让AI"抄作业"：

```
将以下中文翻译成口语化的英文，注意语气要自然：

例子1：吃了吗？→ What's up? / Have you eaten?
例子2：这件事交给我 → I've got this / Leave it to me

现在翻译：你真是太靠谱了
```

### 技巧3：分步提问

复杂任务拆分成多个步骤：

```
第1步：先帮我把这段话分成3个要点
第2步：每个要点用一个比喻来解释
第3步：最后写一个20字的总结
```

### 技巧4：温度（Temperature）控制

- 温度=0：每次回答都一样，严谨、准确
- 温度=0.7：有些变化，创意适中
- 温度=1.0：高创意（写诗、写故事）
- 温度越大，AI越"敢想"

不同场景用不同温度：
- **事实类**（翻译、总结、代码）：温度0-0.3
- **常规写作**：温度0.5-0.7
- **创意写作**：温度0.8-1.0

*注：不同AI工具对温度的称呼可能不同，有的叫"创意度"或"多样性"。*

## 五、Prompt工程实战模板

### 万能Prompt模板
```
角色：[你是谁？AI扮演谁？]
任务：[具体要做什么？]
上下文：[背景信息]
格式：[想要什么形式的结果？]
约束：[有什么限制和要求？]
目标：[为什么要做这个？]
```

### 举例：写一份周报
```
你是一位有5年经验的运营主管。
请根据以下内容帮我写一份周报：
本周完成了用户增长方案设计、数据分析报告、团队周会组织三个方面的工作。
格式：用Markdown，分点列出，每个点包含"做了什么"和"取得的效果"。
语气：专业简洁，不要废话。
```

## 六、本课小结

- Prompt是给AI的指令，好的Prompt = 好的回答
- 四大要素：角色、任务、格式、约束
- 进阶技巧：思维链、Few-shot、分步提问、温度控制
- 可以把常用的Prompt保存成"模板"，以后复用

> 💡 **金句：AI的能力上限取决于它的模型，AI的能力下限取决于你的Prompt。**
