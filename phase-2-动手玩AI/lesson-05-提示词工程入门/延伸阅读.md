## 🎯 实用Prompt模板

**写作类：**
```
你是一位[风格]作家，请写一篇[主题]的[类型]文章，要求[字数]，面向[受众]。
```

**分析类：**
```
请分析[内容]中的关键信息，列出优缺点，给出改进建议，用表格呈现。
```

**学习类：**
```
你是[学科]老师，请用简单语言解释[概念]，举3个生活中的例子，最后出2道自测题。
```
