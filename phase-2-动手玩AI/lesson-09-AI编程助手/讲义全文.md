## 一、编程思维 ≠ 写代码

很多人觉得编程很难，其实难的不是"写代码"，而是"拆解问题"。

**编程的本质：**
1. 把一个大问题拆成很多小问题
2. 每个小问题都能被计算机解决
3. 把小问题的解决方案组合起来

以前，你还需要会"写代码"这个环节。现在有了AI，你只需要会"拆解问题"和"说人话"就可以了。

## 二、主流AI编程工具

### GitHub Copilot
- 装在VS Code里，写代码时自动补全
- 最成熟的AI编程辅助工具
- 适合有编程基础的人

### Cursor
- 专为AI编程设计的编辑器
- 支持自然语言编辑：选中代码 → 按Ctrl+K → 说你想怎么改
- 适合：编程新手和资深开发者

### Claude Code
- 终端里对话式编程
- 可以直接在终端里和Claude对话
- 适合快速原型开发

### ChatGPT/Claude对话模式
- 直接和AI聊天就能写代码
- 适合非程序员偶尔需要写脚本
- 零门槛

## 三、自然语言编程示例

### 示例1：数据分析
```
用户：我有一个Excel文件，里面有一列"销售额"和一列"日期"。
请帮我写一个Python脚本，读取这个文件，计算每个月的总销售额，
并画成柱状图保存为图片。
AI：生成完整的Python代码 + 使用说明
```

### 示例2：文件整理
```
用户：请帮我写一个脚本，我的下载文件夹很乱，希望能按文件类型
把文件自动归类到不同的子文件夹（图片放到Images，文档放到Docs等）
AI：生成脚本 + 使用教程
```

### 示例3：网页
```
用户：帮我做一个简单的个人名片网页，包含头像、姓名、简介、
社交媒体链接。设计要简约好看。
AI：生成完整的HTML/CSS代码
```

## 四、如何给AI清晰的编程需求

写编程Prompt的技巧：

1. **说清楚输入是什么** — 你提供什么数据
2. **说清楚输出是什么** — 你想要什么结果
3. **说清楚环境** — 用的是Windows还是Mac
4. **有问题就贴错误信息** — 直接复制粘贴
