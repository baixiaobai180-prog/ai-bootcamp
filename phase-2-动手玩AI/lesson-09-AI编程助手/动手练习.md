# 动手练习 · 第09课

## 练习一：写第一个小程序（难度：⭐）

**任务：** 给AI以下Prompt，生成一个Python脚本并运行：

```
请帮我在桌面上创建一个"我的日记"文本文档，里面写入今天的日期和一段话："我今天学会了用AI写程序！"
```

## 练习二：Excel处理（难度：⭐⭐）

**任务：** 创建一个简单的Excel文件（含一些数据），让AI帮你处理：

1. 在ChatGPT/Claude上传Excel文件
2. 问："帮我统计每列的平均值、最大值、最小值"
3. 按AI的指示操作

## 练习三：网页制作（难度：⭐⭐⭐）

**任务：** 请AI帮忙制作一个简单的个人主页。

**Prompt参考：**
```
请帮我写一个完整的HTML文件，包含：
- 我的名字（你自己填）
- 一段自我介绍
- 我的三个爱好（图标+文字）
- 社交媒体链接
风格：简约现代，用渐变色，手机适配
```
