# 第09课 · AI编程助手：不会代码也能写程序

## 课件大纲

### 一、编程思维≠写代码（10分钟）
- 编程的本质是"拆解问题"
- AI让编程门槛降到了"会说就能写"

### 二、主流的AI编程工具（20分钟）
- GitHub Copilot：VS Code内置
- Cursor：AI优先的编辑器
- Claude Code：终端对话式编程
- ChatGPT/Claude对话模式

### 三、自然语言编程（20分钟）
- 用中文描述需求→AI生成代码
- 示例：数据分析脚本
- 示例：自动整理文件
- 示例：简单网页

### 四、AI辅助调试（15分钟）
- 把错误信息发给AI
- AI来解释和修复
- 零基础也能自己debug

### 五、实用小项目分享（15分钟）
- 自动处理Excel
- 批量重命名文件
- 网页爬虫
- 个人博客
