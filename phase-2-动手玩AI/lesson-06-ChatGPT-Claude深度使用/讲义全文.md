## 二、深度使用技巧

### 1. 系统提示词（System Prompt）
这是很多用户忽略的最强功能：

**在ChatGPT中，点击左下角头像 → Customize → 设置"我是什么样的用户"**

示例系统提示词：
```
你是一位专业的技术顾问，说话简洁直接，避免不必要的客套。
回答问题时，优先给出结论，再解释原因。
如果遇到不确定的信息，要明确说"不确定"，不要编造。
```

### 2. 多轮对话管理
- **新对话 vs 续对话**：新话题建议开新对话，避免上下文污染
- **定期整理**：重要的对话可以重命名保存
- **上下文窗口管理**：长对话中提醒AI之前的结论

### 3. 文件上传分析
Claude和ChatGPT都支持上传文件：
- **PDF/Word**：让AI读长文档、总结
- **图片**：让AI看图表、识别截图
- **代码**：让AI调试代码
- **Excel**：分析数据

### 4. 联网搜索的正确用法
- 别问"今天天气"，直接说"搜索：北京今天天气"
- 查新闻/事实时一定要开启搜索
- 对比信息可以让AI汇总多个来源

## 三、实用场景模板

### AI当老师的Prompt
```
你是一位[学科]老师。我有[基础水平]。请用苏格拉底式提问法教我[知识点]，
不要直接给答案，而是通过提问引导我自己思考。
```

### AI当编辑
```
请帮我修改这篇文章。要求：
1. 指出逻辑问题
2. 优化语言表达（更简洁/更生动）
3. 增加过渡句
4. 用修订模式标注修改建议
```

### AI做翻译
```
你是一位资深中英翻译。请翻译以下内容：
1. 直译（保留原文结构与语气）
2. 意译（自然的中文表达）
3. 关键术语解释
```
