## 一、AI写文档

### 从零开始写
```
Prompt：请为我写一份「Q2产品运营报告」的大纲，包含：
一、核心数据概览（DAU、转化率、营收）
二、关键动作回顾
三、问题与挑战
四、Q3规划
语气：专业、简洁、数据驱动
```

### 改写优化
把一段文字让AI改写成不同风格：
- 正式商务风 → 适合报告和邮件
- 小红书种草风 → 适合社交媒体
- 简洁直接风 → 适合工作沟通

### 扩写与缩写
- 扩写：200字→800字，增加细节
- 缩写：3000字→300字，保留核心

## 二、AI做PPT

**推荐工具：Gamma.app**

操作流程：
1. 输入主题或大纲
2. 选择模板风格
3. Gamma自动生成完整PPT
4. 在线编辑调整

**与ChatGPT配合：**
先让ChatGPT生成PPT大纲文本，再导入Gamma。

## 三、AI处理Excel

### 公式生成
```
用户：我有一个Excel，A列是销售额，B列是成本，我想在C列计算利润率（(A-B)/A）
帮我生成公式
AI：在C2单元格输入：=(A2-B2)/A2
```

### 数据分析
把数据贴给ChatGPT或Claude，让它帮你分析：
- 哪个月销量最高？
- 哪个产品利润最好？
- 有没有异常值？

## 四、AI阅读总结

### 长文档处理
- 把PDF发给Claude（支持200k上下文）
- 让AI提取关键信息

**常用Prompt：**
```
请帮我总结这篇文档：
1. 核心观点（3句话）
2. 关键数据（列出数字）
3. 我的行动建议
4. 还有哪些待解决的问题
```

## 五、AI会议助手

流程：
1. 用飞书/腾讯会议录制
2. 用通义听悟/飞书妙记转文字
3. 把文字给AI总结

**Prompt：**
```
请根据以下会议记录，提取：
1. 会议决策（每条有谁决定）
2. 待办事项（谁在什么时间前做什么）
3. 争议点（哪些问题还没定论）
4. 下次会议需要讨论什么
```
