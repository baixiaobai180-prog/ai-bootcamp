## 一、AI视频生成

### 主流工具

**Sora（OpenAI）**
- 目前最强的AI视频生成工具
- 能生成长达1分钟高画质视频
- 物理世界理解力最强

**可灵AI（快手）**
- 国产最强，支持文生视频和图生视频
- 免费额度

**Runway Gen-3**
- 最成熟的AI视频工具之一
- 功能多：文生视频、图生视频、修图、手势控制

**Luma Dream Machine**
- 快速生成，效果不错

### Prompt技巧

写视频Prompt和写图片Prompt不太一样——你要描述"发生了什么"而不是"长什么样"：

```
❌ "一个漂亮花园"（静态）
✅ "镜头缓慢推进，穿过一个法式花园，玫瑰花在微风中摇曳，阳光透过树叶洒下斑驳光影，蝴蝶在花丛中翩翩起舞，4K画质"（动态）
```

### 关键技巧：
1. **描述动作** — 动词要用好
2. **镜头语言** — "推/拉/摇/移/升/降"
3. **控制时长** — 太长了AI容易出错
4. **保持简单** — 主体别太多

## 二、AI音乐生成

### Suno
输入歌词和风格描述，Suno就能生成一首完整的歌曲（包括演唱）。

**示例Prompt：**
```
风格：流行民谣，原声吉他，中速
歌词：
[Verse]
清晨的阳光洒在窗前
一杯咖啡在手间
生活虽平凡却温暖
每一天都是新的冒险...
```

### 创作建议
- 先写词再定风格
- 多生成几个版本对比
- 可以组合不同段落

## 三、AI配音

最常用的是**ElevenLabs**：
- 语音极其自然逼真
- 支持情感控制
- 支持语音克隆

## 四、短视频制作全流程

用AI从零制作一个30秒短视频：

1. **剧本** — Claude/ChatGPT写脚本
2. **配音** — ElevenLabs生成旁白
3. **视频画面** — 可灵AI/ Runway生成
4. **背景音乐** — Suno生成
5. **剪辑** — CapCut/剪映组合导出
