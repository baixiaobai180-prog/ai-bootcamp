# 第08课 · AI视频与音乐：多模态内容创作

![配图](assets/images/lesson-08-ai-video-music.jpg)


## 课件大纲

### 一、AI视频生成（25分钟）
- 主流工具：Sora / 可灵AI / Runway / Luma / Pika
- 文本到视频（Text-to-Video）
- 图片到视频（Image-to-Video）
- Prompt技巧：动作描述+场景+镜头

### 二、AI音乐生成（20分钟）
- Suno / Udio：文本到完整歌曲
- 写词+定风格+生成
- AI音乐创作的局限

### 三、AI配音与语音（15分钟）
- ElevenLabs：最逼真的AI配音
- FishAudio：开源的语音合成
- 语音克隆技术

### 四、多模态组合创作（20分钟）
- AI写剧本 + AI配音 + AI视频 + AI配乐
- 短视频制作全流程
