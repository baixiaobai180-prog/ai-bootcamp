# 第07课 · AI绘画入门：用文字创造图像

## 一、AI绘画是什么？

AI绘画就是**用文字描述来生成图像**。你输入一句话，AI就给你画出一张图。

背后的核心技术是**扩散模型（Diffusion Model）**：

通俗理解：AI先学习"如何把一张图慢慢加噪变成纯噪点"，然后反过来——从一个纯噪点图，一步步去噪，最终还原成一张清晰的图片。

2022年，Stable Diffusion开源引发AI绘画革命；2024年，Flux模型将画质推到了新高度。

## 二、主流工具对比

### Midjourney
- 艺术风格最强，出片质量高
- 用Discord操作，界面独特
- 适合：艺术创作、商业设计
- 费用：$10-60/月

### DALL-E 3
- OpenAI出品，内置在ChatGPT里
- 最"听话"——对文字的遵循度最高
- 适合：新手入门、快速出图
- 费用：ChatGPT Plus用户可用

### Stable Diffusion
- 完全免费开源
- 可本地跑（需要好显卡）
- 可玩性最高：LoRA、ControlNet
- 适合：技术用户、想深度定制

### Flux
- Black Forest Labs出品（原Stable Diffusion团队）
- 2024年发布，画质超越SD3和Midjourney
- 开源版可用
- 适合：追求最高画质的用户

## 三、写Prompt的万能公式

**AI绘画的Prompt质量 = 图像质量。**

万能公式：
```
[主体] + [动作/姿态] + [场景/环境] + [风格] + [细节] + [光线/氛围] + [视角/构图]
```

**示例：**
```
主体：一只橘猫
动作：慵懒地躺在窗台上
场景：午后阳光照进房间
风格：吉卜力动画风格
细节：毛茸茸的尾巴，树影斑驳
光线：暖色夕阳，柔光
视角：特写，浅景深
```

**对比：**
- ❌ "一只猫" → 普通图片
- ✅ 用上面的完整Prompt → 电影级的画面

### 常用风格关键词
| 风格 | 关键词（中文提示） |
|------|-----------------|
| 写实 | 照片级真实感、8K、超高清 |
| 卡通 | 动画风格、迪士尼、吉卜力 |
| 油画 | 油画风格、梵高、莫奈 |
| 3D | 3D渲染、皮克斯、C4D |
| 赛博朋克 | 霓虹灯、未来城市、雨夜 |
| 水墨画 | 水墨风格、留白、中式美学 |

### 负面提示词（Negative Prompt）
告诉AI不要画的内容：
```
模糊、丑陋、变形、多余的手、多余的手指、低质量、水印、文字
```

## 四、常用参数

| 参数 | 作用 | 推荐值 |
|------|------|--------|
| Aspect Ratio | 画面比例 | 1:1(方图)/3:4(竖图)/16:9(横图) |
| Stylization | AI创意度 | 50-100(默认)/100-250(更有艺术感) |
| Seed | 种子值 | 不填=随机，填了=可复现 |
| Steps | 生成步数 | 20-50(步数越高细节越多) |

## 五、实战：常见设计场景

### 场景1：社交媒体配图
```
Prompt: 一个极简风格的办公桌，一台MacBook，一杯咖啡，一盆绿植，
柔和自然光，俯拍，ins风摄影，温暖舒适的氛围
```

### 场景2：Logo设计
```
Prompt: 一个科技公司的logo，简约设计，蓝色渐变，几何图形，
圆形构图，扁平化风格，白底
```

### 场景3：商业海报
```
Prompt: 促销海报，红色背景，白色商品居中摆放，
金色字体"大促销"，简洁大气，商业摄影，高级质感，阴影效果
```

## 六、版权与伦理

- 不要用AI生成他人风格的作品进行商业销售
- 不要生成虚假人物/内容
- 注意平台规定：不同平台对AI绘画的使用限制不同
- AI绘画的版权归属：各平台规定不同，出图前了解清楚
